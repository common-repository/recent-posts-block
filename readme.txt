Gutenberg Recent Posts.

== Description ==
The Recent Posts Block for Gutenberg lets you add a recent posts section in your posts or pages. It provides support for your custom post types. simply select post type from list of post types for which you want to display recent posts. It also comes with customization options like image size, post meta options, ordering etc. to suit your needs.

= Features =
* Get recent posts of your favourite custom post type.
* Manage ordering like ascending or descending.
* Image size options.
* Post meta settings like post date, post excerpt etc.

== Installation ==
1. Upload plugin to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.

== Screenshots ==
1. Open add new blocks from gutenberg editor and go to widgets and add recent posts block by just clicking on it.
2. Select post type from list of post types for which you want to display recent posts.
3. Select required settings from sidebar like number of posts, ordering etc.
